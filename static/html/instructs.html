Instructions

About Game

Memory Card Game is game where you have to remember the cards which you open. If you open card with same symbol certain number of times, you will solve that cards. When you solve all cards, you win the game.

Singleplayer

Singleplayer mode is game where you have to solve all cards, alone. There is five modes of singleplayer game and they are: easy, medium, hard, extreme and custom. For every game you play software will measure your time and your moves. Less time and less moves are better. You can send your score to the server and see later it on the score board.

Easy mode is game where you have grid with 4x4 cards. For solving cards you have to open two cards with same symbol. There is 8 different symbols in this game.

Medium mode is game where you have grid with 6x6 cards. For solving cards you have to open three cards with same symbol. There is 9 different symbols in this game.

Hard mode is game where you have grid with 8x8 cards. For solving cards you have to open four cards with same symbol. There is 10 different symbols in this game.

Extreme mode is game where you have grid with 12x12 cards. For solving cards you have to open three cards with same symbol. There is 10 different symbols in this game. After you solve one set of cards, all cards will be shuffled. While cards are shuffling, time will be paused.

Multiplayer

Multiplayer mode is game where you have to solve all cards, with another player. One player open card, and if another player open card with same symbol, cards will be solved. One player have his own grid and another player have his own grid, size of grid is 4x4.
